About This Project

Student Finance Tracker - A comprehensive expense management tool

Overview

The Student Finance Tracker is a web-based application designed to help students manage their finances effectively. Built with vanilla HTML, CSS, and JavaScript, it demonstrates modern web development practices while maintaining simplicity and accessibility.

Key Features

  • Transaction Management: Add, edit, delete, and view all your financial transactions
  • Smart Search: Use regex patterns to search and filter transactions
  • Budget Tracking: Set monthly budget caps and monitor your spending
  • Statistics Dashboard: View spending trends and category breakdowns
  • Multi-Currency Support: Work with USD, EUR, and GBP with built-in converter
  • Data Portability: Import and export your data as JSON
  • Offline Storage: All data stored locally in your browser

Technical Implementation

Technologies Used

  • HTML5 with semantic markup
  • CSS3 with custom properties and responsive design
  • Vanilla JavaScript (ES6+ modules)
  • localStorage API for data persistence
  • Regular expressions for validation and search

Validation Rules

The application implements comprehensive regex-based validation:

  • Description: 3-100 characters, alphanumeric with basic punctuation
  • Amount: Decimal format (0.00 to 9999.99)
  • Date: ISO format (YYYY-MM-DD)
  • Category: Predefined list validation
  • Duplicate Detection: Advanced regex with back-references

Accessibility Features

This application is built with accessibility as a priority:

  • WCAG 2.1 Level AA compliant
  • Full keyboard navigation support
  • ARIA labels and live regions
  • Semantic HTML structure
  • High contrast color scheme (4.5:1 minimum)
  • Screen reader friendly
  • Skip navigation links

Keyboard Shortcuts

Shift + Tab Navigate backward
